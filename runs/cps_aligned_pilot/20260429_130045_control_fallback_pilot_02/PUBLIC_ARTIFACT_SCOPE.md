# Public artifact scope

This public copy keeps the run note, status file, summary, and selected evidence extracts.

Raw Home Assistant state snapshots, raw ESPHome runtime captures, local machine dumps, device-network identifiers, and environment-specific configuration are not included in this public repository.

This repository is intended to show the pilot structure, evidence boundary, and selected observations without exposing the local home-automation environment.

Full raw captures are retained only in the non-public working archive.
