## Run folder structure

Each accepted run folder contains:

- `run_note.md`
- `RUN_STATUS.txt`
- `PUBLIC_ARTIFACT_SCOPE.md`
- `summary/run_summary.md`
- selected files under `evidence_extracts/`

## Boundary

The accepted runs are used as initial pilot evidence.

They do not claim long-term stability, physical heating performance, or comparison between recovery designs.
